# SwiftDraw

A Swift library for parsing and drawing SVG images to CoreGraphics contexts.

## Usage

## iOS

```swift
import SwiftDraw
let image = UIImage(svgNamed: "sample.svg")
```

## macOS

```swift
import SwiftDraw
let image = NSImage(svgNamed: "sample.svg")
```
